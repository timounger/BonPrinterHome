Report

Note: only available for Admin and Host user

Show Status

In the menu bar Prints → Show Status you can display the current status of printouts. Menu is not available for empty log list. Service Stuff users can only view their own reports.

Staff users can also only view their status.

Open Folder

In the menu bar Prints → Open Folder the output location is opened.

Combin Report

In the menu bar Prints → Report you can combine multiple reports into a single file.

Create Report

In the menu bar Prints → Create Report a complete report of printouts generated. Menu is not available for empty log list.

The printout logs are deleted.
